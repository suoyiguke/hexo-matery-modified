MySQL Performance 这应该是每个 MySQL DBA 必上的博客； Planet MySQL MySQL 官方出品； Dba Square 非常不错的博客； High Availability MySQL 了解 MySQL 高性能； Baron Schwartz 「高性能 MySQL」 作者的博客； Ronald Bradford 「Effective MySQL」 系列作者的博客； Yoshinori Matsunobu 博主是 Facebook 数据库工程师。 NoSQL Notes 阿里日照前辈的博客； InsideMySQL 「MySQL 技术内幕：InnoDB 存储引擎」、「MySQL 技术内幕：SQL 编程」、「MySQL 内核：InnoDB 存储引擎 卷1」作者姜承尧前辈博客； DBA Notes 冯老师的博客，但现在的重心不是数据库了； AnySQL 楼方鑫前辈的博客； Sky Jian 「MySQL 性能优化与架构设计」作者简朝阳前辈的博客； NinGoo 阿里巴巴 DBA 宁海元前辈博客； yejr MySQL 中文网站长叶金荣前辈博客； orczhou 淘宝 MySQL DBA 周振兴前辈博客； Focus on MySQL/InnoDB kernel 阿里巴巴 MySQL 内核开发工程师郭鹏前辈的博客。 其他网站

MySQL 官网 这个网站再熟悉不过了吧； ITPub 这个网站在国内很有名，上面孕育了一大批杰出的数据库大牛； GitHub 我认为这是 IT 从业者必上的网站； Stack Overflow 相信平时遇到的问题都可以在上面找到不错的答案； 开源中国 国内首屈一指的网站，为开源事业做了巨大贡献； InfoQ 很多分享，值得关注的网站； WooYun 关注安全的好去处； 知乎周刊 充电的好网站； 码农周刊 还是应该关注编程； Startup News 冯老师做的； 极客头条 关注互联网动态。 书籍
